## [0.1.0] - 2026-07-14

### Features

Initial release of AgentPack — a unified MCP / Skills / Agent management desktop application for AI coding tools.

#### Agent Management

- Auto-detect installed AI coding tools (Claude Code, Codex, Cursor, OpenCode, Trae, etc.)
- Enable / disable individual agents, persistent disabled state
- Smart cleanup: undetected agents are marked first, then auto-removed

#### MCP Server Management

- Full CRUD for MCP servers, supporting stdio / SSE / HTTP protocols
- Multi-agent binding: one MCP server can be bound to multiple agents with auto-config write
- One-click scan of agent config directories to discover unmanaged MCP servers
- Shell command injection prevention via security validation
- Automatic deduplication, database sync, and rollback on failure

#### Skills Management

- Install / uninstall Skills from GitHub repositories or ZIP files
- Multi-agent binding: sync a Skill to multiple agents' skill directories
- Two sync modes: symlink (default, space-saving) and copy
- Two storage locations: centralized and unified directory
- Auto-adoption: scan agent skill directories and take over unmanaged Skills
- Update detection: compare local state with remote repository to find updates

#### Skill Marketplace

- Multiple market sources: Official Registry, Smithery, skills.sh, custom GitHub repos
- Cross-source search results with automatic deduplication, sorted by popularity
- 24-hour caching and singleflight to prevent duplicate requests

#### Configuration Backup & Restore

- Automatic snapshots on MCP server changes
- Manual backup with descriptions
- Export to file and cross-device import & restore
- Configurable backup retention count

#### Settings & Personalization

- Light / Dark / System themes (Mica effect support on Windows)
- Independent market source toggles
- Window close behavior: prompt / minimize to tray / quit
- Backup strategy configuration

#### Others

- System tray support, minimize to background
- Cross-platform: Windows, macOS, Linux
- Multi-agent config format support (JSON, TOML)
- Encrypted environment variable storage
- SQLite database persistence
